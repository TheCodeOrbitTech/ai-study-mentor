# AI Study Mentor

## Overview
AI Study Mentor is a multi-agent AI system that helps students:

- Create personalized study plans
- Find learning resources
- Track study progress
- Get career guidance
- Receive motivation and reminders

## Problem Statement
Students often struggle with planning, consistency, and finding the right learning resources. This project provides an AI-powered mentor that guides students throughout their learning journey.

## Solution
The system uses multiple AI agents working together:

1. Study Planner Agent
2. Resource Finder Agent
3. Progress Tracker Agent
4. Career Guidance Agent
5. Motivation Agent
